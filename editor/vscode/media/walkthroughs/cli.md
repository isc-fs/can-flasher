# The `can-flasher` CLI

Everything this extension does — flashing, discovery, diagnostics — runs through the **`can-flasher`** command-line tool. You don't have to install it by hand: the extension downloads a version-matched copy automatically.

Run **Doctor** to confirm the CLI resolved and matches this extension's version. It also checks your adapter, workspace, node-id, and firmware artifact in one pass — the fastest way to know you're ready to flash.

> Prefer your own build on `$PATH`? Run **ISC MingoCAN: Use CLI on PATH (ignore managed download)**.
